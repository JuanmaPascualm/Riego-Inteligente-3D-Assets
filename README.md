# 🌿 Riego Inteligente IoT - Almería 2026 🚀

Este repositorio contiene los activos de diseño industrial para el sistema de riego automatizado basado en el ecosistema ESP32/ESP8266. El diseño se centra en la **modularidad**, la **seguridad electrónica** y el **mantenimiento**.

---

## 📂 Contenido del Repositorio
* **`STL/`**: Archivos listos para el software de laminado (Cura, PrusaSlicer, etc.).
* **`GCODE/`**: Archivos optimizados para impresión directa.
* **`Inventor/`**: Archivos fuente (.ipt, .iam) para modificaciones en Autodesk Inventor.

---

## 🛠️ Especificaciones de la Carcasa
El diseño consta de **3 gabinetes independientes** para garantizar la estanqueidad y seguridad del sistema:

1.  **🔵 Gabinete de Control**: Aloja la pantalla E-Paper y la electrónica de potencia.
2.  **🟢 Gabinete de Cultivo**: Espacio aislado para la planta y el sensor de humedad.
3.  **💧 Gabinete Hidráulico**: Depósito de agua con bomba sumergible integrada.

---

## ⚙️ Configuración de Impresión Recomendada
| Parámetro | Valor Sugerido |
| :--- | :--- |
| **Material** | PLA o PETG (mejor resistencia a humedad) |
| **Relleno (Infill)** | 15% - 20% (Giroide para mayor fuerza) |
| **Soportes** | Solo necesarios en el Gabinete de Control |
| **Capa** | 0.2mm o 0.28mm (Draft) |

---

## 🔌 Doble Alimentación
El diseño es versátil y permite al usuario elegir según la necesidad:
* **🔋 Portabilidad**: Espacio para Pila de 9V (mantenimiento preventivo).
* **🔌 Estabilidad**: Conector para fuente externa de **12V 2A** (recomendado para evitar ruidos eléctricos).
